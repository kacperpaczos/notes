## Types of UML Diagrams

UML diagrams are divided into two main categories:

### Structural Diagrams
- Class Diagrams
- Component Diagrams
- Object Diagrams
- Deployment Diagrams
- Package Diagrams

### Behavioral Diagrams
- Sequence Diagrams
- Activity Diagrams
- Use Case Diagrams
- State Diagrams
- Communication Diagrams
- Timing Diagrams
- Interaction Overview Diagrams
